# 온라인 정육점 구현 프로젝트_고기샵

## 개요
- 온라인 사이트 구현 프로젝트
- 기술 / 환경
  - JAVA
  - JSP
  - HTML
  - CSS
  - JavaScript
- 기간: 2021.04.19 ~ 2021.04.27 (7일)

## 내용
- 실제 온라인 정육점 사이트를 구현하였습니다.
  - 메인 페이지
  - 상품 전체보기 페이지
  - 상품 상세보기 페이지
    - 제품정보
    - 제품 상세정보 탭
    - 리뷰 탭
    - 상품 기본정보 탭
  - 마이 페이지 
  - 결제 페이지
  - 장바구니 페이지
- 회원가입 후 상품을 주문할 수 있는 사이트를 제작하였습니다.
